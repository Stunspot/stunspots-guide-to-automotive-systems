# Status

*Stunspot's Guide to Automotive Systems* is a **v1.0 first-pass public canon release** built from an existing AI-assisted automotive knowledge corpus.

The repository is suitable for model-facing use as a structured reference corpus, RAG substrate, long-context knowledge source, and human-readable field manual. It should still be treated as a knowledge artifact requiring verification for safety-critical, legal, financial, regulatory, or repair-specific use.

## Corpus Maturity

The corpus is structurally complete for this release: reports A-O are present as individual Markdown source reports, and the repository includes public navigation, citation metadata, a manifest, a changelog, and AI/RAG-oriented usage guidance.

This is not represented as a final textbook edition, certified technical manual, OEM service source, legal compliance guide, or professional repair authority. It is a systems-oriented knowledge canon for reasoning support.

## Packaging Notes

- Source reports in `knowledge-packs/by-report/` are the canonical individual units.
- Compiled packs in `knowledge-packs/compiled-packs/` are convenience upload formats.
- The omnibus file in `knowledge-packs/omnibus/` is a whole-corpus convenience bundle.
- This release includes compiled packs for A-D, E-J, and N-O.
- Reports K-M are available as source reports and included in the omnibus; they are not currently packaged as a separate compiled Vol. 3 bundle.
- `docs/` contains navigation and GitHub Pages files, not the individual source-report corpus.
